# ArbiSecure Frontend

The frontend for **ArbiSecure**, a trustless service escrow platform built on Arbitrum Stylus.

## Overview
ArbiSecure provides a seamless, "Link-based" experience for global freelance payments. It leverages Arbitrum's speed and low fees to offer instant settlement and trustless escrow services.

## Tech Stack
-   **Framework:** [Next.js 15](https://nextjs.org/) (App Router)
-   **Styling:** Tailwind CSS with custom Design System
-   **Animations:** Framer Motion
-   **Fonts:** Inter, Playfair Display, Space Mono

## Getting Started

1.  **Install dependencies:**
    ```bash
    npm install
    ```

2.  **Run the development server:**
    ```bash
    npm run dev
    ```

3.  **Open the app:**
    Visit [http://localhost:3000](http://localhost:3000)

## Project Structure
-   `app/` - Next.js App Router pages and layouts.
-   `components/` - Reusable UI components.
    -   `landing/` - Landing page specific sections.
    -   `animations/` - Framer Motion animation components.
-   `public/` - Static assets.

## Key Features
-   **Glassmorphism UI:** Premium, modern aesthetic.
-   **Scroll Animations:** Awwwards-style reveal effects.
-   **Trustless Security:** Integration with Stylus smart contracts.
